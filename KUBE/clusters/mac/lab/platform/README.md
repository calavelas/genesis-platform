# Lab Platform Apps

Platform ArgoCD child `Application` manifests for `mac/lab` live here.

Current usage:
- `argocd.yaml`: ArgoCD self-management app.
- `traefik.yaml`: Traefik platform app.
- `endr-api.yaml`: ENDR backend API app (Helm chart at `ENDR/charts/endr-api`).
- `endr-case.yaml`: ENDR frontend app (Helm chart at `ENDR/charts/endr-case`).
